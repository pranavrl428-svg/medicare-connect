# 📖 About the Project

**MediCare Connect** is a healthcare management mobile application developed using **Flutter** and **Firebase**. It helps patients manage healthcare services digitally by allowing them to register, log in, book appointments, view prescriptions, request medicines, manage profiles, and submit follow-up requests.

The project was developed as an internship project to demonstrate modern mobile application development using Flutter and Firebase.

# 🚀 Installation

Clone the repository

```bash
git clone <YOUR_GITHUB_REPOSITORY_LINK>
```

Navigate to the project

```bash
cd medicare_connect
```

Install dependencies

```bash
flutter pub get
```

Run the application

```bash
flutter run
```
